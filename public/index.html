Reference Services

Public SSH keys API

Why

I created this API because I wanted to update the access keys of my server with my Github keys. Once there was an API endpoint to get the keys of a user in text form but that disappeared. So I created this API to get those keys. While doing that I also export the keys in other formats.

Reference

All api urls can return different formats:

  • text (default)
  • json
  • xml

Select what format you want:

Using the format query parameter:

GET https://public-keys-service.herokuapp.com/api/services?format=text HTTP/1.1

GET https://public-keys-service.herokuapp.com/api/services?format=json HTTP/1.1

GET https://public-keys-service.herokuapp.com/api/services?format=xml HTTP/1.1

Using Accept header:

GET https://public-keys-service.herokuapp.com/api/services HTTP/1.1
Accept: text/text

GET https://public-keys-service.herokuapp.com/api/services HTTP/1.1
Accept: application/json

GET https://public-keys-service.herokuapp.com/api/services HTTP/1.1
Accept: text/xml

GET https://public-keys-service.herokuapp.com/api/services HTTP/1.1
Accept: application/xml

Get all available services

GET https://public-keys-service.herokuapp.com/api/services HTTP/1.1

Get keys for a user from a specific service

GET https://public-keys-service.herokuapp.com/api/keys/github/thekingdave HTTP/1.1

Get keys for a user from default service

GET https://public-keys-service.herokuapp.com/api/keys/thekingdave HTTP/1.1
